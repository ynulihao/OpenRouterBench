# Benchmark Results Download

This directory contains LLMRouterBench benchmark results data.

## Download Instructions

To obtain the complete benchmark results, download the compressed file from the cloud storage and extract it to the current directory (`results/`).

After extraction, the directory structure should be:
```
results/
  └── bench/
      ├── aime/
      ├── bbh/
      ├── humaneval/
      └── ...
```

## Extraction Command

```bash
# Extract to current directory
tar xzf bench.tar.gz
```
